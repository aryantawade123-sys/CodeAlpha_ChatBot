# 🤖 Java AI Chatbot

## 📌 Project Description

This is a Java-based AI chatbot developed using NLP techniques and GUI.

## 🚀 Features

* NLP-based keyword matching
* FAQ training system
* Interactive chat interface (GUI)
* User input handling

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* HashMap (Knowledge Base)

## 🎯 How It Works

The chatbot uses keyword matching and a knowledge base to generate responses. It also supports training using custom inputs.
